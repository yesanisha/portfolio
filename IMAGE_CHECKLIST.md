## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Size**: Max 500KB per image (optimize for web)
- **Dimensions**: 1200x800 px (maintains 3:2 aspect ratio)
- **Quality**: High enough to look sharp, compressed enough to load fast

## Tools to Create/Optimize Images

1. **Screenshots**: Use phone/browser screenshot tools
2. **Optimization**: Use [TinyPNG.com](https://tinypng.com) or [Squoosh.app](https://squoosh.app)
3. **Design**: Use Canva/Figma for logo on gradient backgrounds
4. **QR Codes**: Use [QR Code Generator](https://www.qr-code-generator.com)
